Basic HTML tags and what the do
HTML Elements & Structure What it does Closing tag
<!DOCTYPE html> Document declaration,
required first line of an HTML document
<html> The root of and HTML document,
Should be added after the doctype declaration		 Requires closing tag
<head> Contains general info about the page that isnt displayed Requires closing tag
<h1>-<h6> The six differnt levels of heading elements Requires closing tag
<p> Paragraph element contains and displays a block of text Requires closing tag
<body> Represents the content of the HTML document,
there can be only one body element		 Requires closing tag
<title> Contains the text that is diplayed in the browsers title bar Requires closing tag
<ul>-<ol> Orginized(Numbered) & Unorginized lists Requires closing tag
<li> The items you want in your list Requires closing tag
<video> Embeds a media player for video playback
(must use the src="" to link the video in opening tag		 Requires closing tag
<img> Embeds images in your document, must also use the src="" atribute
<em> Emphasis text Requires closing tag
<div> Used as a container to divide your document into sections Requires closing tag
<br> Will create a break in text
<a> Used to create hyperlinks(requires href="url") in opening tag Requires closing tag
<strong> Highlight(bold) the text this tag surrounds Requires closing tag
<span> Used to group text together for styling purposes Requires closing tag
